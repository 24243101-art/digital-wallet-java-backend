This is a console-based Digital Wallet backend project developed using Java during my training period.

## Features
- Add transaction
- View transaction by ID
- Update transaction amount
- Delete transaction
- Sort transactions by ID
- Sort transactions by amount
- File-based storage using transactions.txt

## Technologies Used
- Java
- OOP concepts
- DAO Design Pattern
- File Handling
- Collections
- Exception Handling

## How It Works
- User interacts through a console menu
- Transactions are stored in a text file
- DAO layer handles all business logic
